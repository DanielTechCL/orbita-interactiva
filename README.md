# orbita-interactiva
Programa en Python que permite generar y visualizar una órbita circular de forma interactiva. El usuario define el radio en kilómetros y la cantidad de ángulos, y el sistema clasifica la órbita como baja, media o alta. Ideal para aprender trigonometría, visualización científica y uso básico de NumPy y Matplotlib. para estudiantes y principiantes.!!
