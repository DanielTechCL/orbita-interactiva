# orbita-interactiva
# Órbita Interactiva en Python

Este proyecto es un programa en Python que permite generar y visualizar una órbita circular de forma interactiva. El usuario puede definir el radio de la órbita en kilómetros y la cantidad de ángulos utilizados para el cálculo, obteniendo una representación gráfica clara y a escala.

## Descripción
El programa calcula las coordenadas X e Y de una órbita circular utilizando funciones trigonométricas. Además, clasifica la órbita según su altura sobre el nivel del mar, indicando si se trata de una órbita baja, media o alta, lo que agrega un componente educativo al proyecto.

## Cómo funciona
- El usuario ingresa el radio de la órbita en kilómetros.
- El usuario define la cantidad de ángulos a utilizar para el cálculo.
- El programa determina el tipo de órbita según el radio ingresado.
- Se genera un gráfico en dos dimensiones manteniendo la escala real.

## Características
- Interacción directa con el usuario mediante entrada por consola.
- Clasificación de órbitas: baja, media y alta.
- Visualización clara y proporcional de la órbita.
- Código simple y fácil de entender, ideal para principiantes.

## Tecnologías
- Python 3
- NumPy
- Matplotlib

## Objetivo del proyecto
Este proyecto tiene fines educativos y está pensado para personas que están aprendiendo Python, matemáticas básicas y visualización científica, permitiendo comprender de forma simple cómo se modelan y representan órbitas circulares.



# Interactive Orbit in Python

This project is a Python program that allows users to generate and visualize a circular orbit in an interactive way. The user can define the orbit radius in kilometers and the number of angles used for the calculation, obtaining a clear and scaled graphical representation.

## Description
The program calculates the X and Y coordinates of a circular orbit using trigonometric functions. It also classifies the orbit according to its height above sea level, indicating whether it is a low, medium, or high orbit, adding an educational component to the project.

## How it works
- The user enters the orbit radius in kilometers.
- The user defines the number of angles used for the calculation.
- The program determines the orbit type based on the entered radius.
- A two-dimensional plot is generated while keeping the correct scale.

## Features
- Direct user interaction through console input.
- Orbit classification: low, medium, and high.
- Clear and proportional orbit visualization.
- Simple and beginner-friendly code.

## Technologies
- Python 3
- NumPy
- Matplotlib

## Project goal
This project is for educational purposes and is intended for people who are learning Python, basic mathematics, and scientific visualization, allowing them to understand how circular orbits are modeled and displayed in a simple way.
